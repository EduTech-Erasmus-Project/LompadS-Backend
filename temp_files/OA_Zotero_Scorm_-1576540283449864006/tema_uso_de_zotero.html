Tema: Uso de Zotero

Para ingresar referencias a Zotero tenemos dos opciones, una es trabajando con el programa Zotero de escritorio, que hemos instalado en nuestro computador, a este programa podemos añadir referencias bibliográficas digitando manualmente los nuevos items o registros y añadiéndolos a las diferentes colecciones y bibliotecas que hemos creado. Si la aplicación Zotero de escritorio está abierta, se puede añadir registros directamente desde el navegador en el que hemos instalado el componente, Si la aplicación no está abierta, los registros se añadirán a la base de datos de Zotero en la nube, para esto es necesario registrarse como usuario en el servicio www.zotero.org. También se puede trabajar en www.zotero.org directamente en la aplicación  para añadir y consultar registros. En el vide a continuación veremos como utilizar Zotero.
